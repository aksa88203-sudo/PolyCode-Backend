## 🚀 Quick Start

### Web Interface (Recommended)
```bash
# Navigate to website backend
cd website/backend/

# Install dependencies
npm install

# Start the server
npm start

# Open in browser
# http://localhost:5000
```

### For Developers
```bash
# Navigate to source code
cd data/code/

# Run any project
python projects/task_manager.py

# Run algorithms
python algorithms/search_algorithms.py

# Run utilities
python utilities/password_generator.py
```

### For Learners
```bash
# Start with documentation
cd data/documentation/

# Begin learning journey
cat README.md
```
