# C++ Programming Repository

A comprehensive collection of C++ projects, problems, and learning materials organized into a structured learning path following the Python course structure.

## 📁 Structured Learning Path

The data is organized into topic-based categories to make it easier to follow a logical learning progression.

```
Cplusplus/data/
├── getting_started/              # Introduction to C++, setup, first program
├── concepts/                     # Variables, operators, I/O, functions, pointers
├── data_structures/              # Arrays, strings, structures, unions
├── algorithms/                   # Coming soon - Search, sorting algorithms
├── object_oriented_programming/  # Classes, inheritance, polymorphism
├── advanced_topics/              # Templates, STL, concurrency, networking
├── examples/                     # Coming soon - Practical examples
└── projects/                     # Coming soon - Complete projects
```

- **Key Takeaways** - Summary of important points

## 🛠️ Technologies & Tools
- **Language:** C++ (C++11/14/17/20 standards)
- **Build System:** GCC, Clang, or MSVC
- **Libraries:** STL, SFML (for games)

## 🏁 Installation & Usage
1. **Clone the repository:**
   ```bash
   git clone https://github.com/SENODROOM/PolyCode.git
   cd PolyCode/Cplusplus
   ```
2. **Compile and run a program:**
   ```bash
   g++ -std=c++17 "data/02) Fundamentals/01) IO Statements.cpp" -o output
   ./output
   ```
