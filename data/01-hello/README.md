# Hello World Examples

This directory contains various "Hello World" style programs to introduce Go programming basics.

## Files

- **main.go** - Basic Hello World program
- **variables.go** - Hello with variables
- **input.go** - Interactive Hello with user input
- **multiple.go** - Multiple Hello examples
- **formatted.go** - Formatted Hello with Printf

## Running the Programs

```bash
go run main.go
go run variables.go
go run input.go
go run multiple.go
go run formatted.go
```

## What You'll Learn

1. **Basic Program Structure** - Package main, import, func main()
2. **Printing Output** - fmt.Println(), fmt.Printf()
3. **Variables** - Declaration and initialization
4. **User Input** - Reading from stdin
5. **Data Structures** - Slices and maps
6. **String Formatting** - Format verbs and alignment

## Try This

1. Modify the messages to say something personal
2. Add more variables (height, weight, hobbies)
3. Try different format verbs (%x, %b, %o)
4. Add error handling for user input
5. Create your own Hello variation
