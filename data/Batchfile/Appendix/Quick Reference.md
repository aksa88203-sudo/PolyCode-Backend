## Command Syntax Quick Guide

### Basic Structure
```batch
COMMAND [options] [parameters]
```

### Common Patterns
```batch
ECHO message                  REM Display text
SET variable=value            REM Create variable
IF condition command          REM Conditional
FOR %%var IN (set) DO cmd     REM Loop
CALL label                    REM Subroutine
```

## File Extensions

| Extension | Type | Purpose |
|-----------|------|---------|
| `.bat` | Batch file | Standard batch script |
| `.cmd` | Command file | Windows NT batch |
| `.exe` | Executable | Compiled program |
| `.txt` | Text file | Plain text data |
| `.log` | Log file | Script output logs |

## Exit Codes

| Code | Meaning |
|------|---------|
| `0` | Success |
| `1` | General error |
| `2` | File not found |
| `3` | Path not found |
| `5` | Access denied |
| `9009` | Command not found |
