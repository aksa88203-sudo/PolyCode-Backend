## Quick Start

Batch files are text files with a `.bat` or `.cmd` extension that contain a series of commands executed by the Windows Command Processor (cmd.exe).

### Creating Your First Batch File

1. Open Notepad or any text editor
2. Type your batch commands
3. Save the file with a `.bat` extension
4. Double-click to run it

### Example: Hello World

```batch
@echo off
echo Hello, World!
pause
```

## Key Concepts

- **@echo off** - Prevents commands from being displayed
- **echo** - Displays messages
- **rem** - Adds comments
- **pause** - Waits for user input
