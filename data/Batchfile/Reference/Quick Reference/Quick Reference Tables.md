## Special Variables

| Variable | Value | Example |
|----------|-------|---------|
| `%0` | Batch file name | `%0` |
| `%1-%9` | Parameters | `%1` |
| `%*` | All parameters | `%*` |
| `%CD%` | Current directory | `%CD%` |
| `%DATE%` | Current date | `%DATE%` |
| `%TIME%` | Current time | `%TIME%` |
| `%RANDOM%` | Random number | `%RANDOM%` |
| `%ERRORLEVEL%` | Error code | `%ERRORLEVEL%` |

## FOR Loop Modifiers

| Modifier | Result |
|----------|--------|
| `%%~fI` | Full path |
| `%%~dI` | Drive letter |
| `%%~pI` | Path only |
| `%%~nI` | Filename |
| `%%~xI` | Extension |
| `%%~zI` | File size |
