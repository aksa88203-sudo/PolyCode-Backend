A comprehensive learning guide for C# programming with practical examples and exercises.

- [Setting up Development Environment](docs/02-setup.md)
- [Your First C# Program](docs/03-first-program.md)

### 📝 Module 2: Basic Syntax & Concepts

- [Operators and Expressions](docs/05-operators.md)
- [Control Flow](docs/06-control-flow.md)
- [Methods and Functions](docs/07-methods.md)

### 🏗️ Module 3: Object-Oriented Programming

- [Inheritance](docs/09-inheritance.md)
- [Polymorphism](docs/10-polymorphism.md)

### 🔧 Module 4: Intermediate Concepts

## 🛠️ Prerequisites

- Basic understanding of programming concepts
- Windows, macOS, or Linux operating system
- .NET SDK (version 6.0 or later)

## 🚀 Getting Started

1. Clone this repository
2. Navigate to the setup guide: [Setting up Development Environment](docs/02-setup.md)
3. Follow the installation instructions
4. Start with [Your First C# Program](docs/03-first-program.md)

## 🎯 Learning Outcomes

After completing this guide, you will be able to:

- Write clean, efficient C# code
- Design and implement object-oriented solutions
- Work with collections and data structures
- Handle errors and exceptions gracefully
- Build various types of applications
- Use modern C# features and best practices
