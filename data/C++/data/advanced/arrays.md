# Arrays in C

Arrays store multiple elements of the same type in contiguous memory.

## Example
```c
#include <stdio.h>

int main() {
    int numbers[5] = {1, 2, 3, 4, 5};
    printf("%d\n", numbers[2]); // prints 3
    return 0;
}
