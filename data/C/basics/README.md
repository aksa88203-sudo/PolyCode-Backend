# Basics

## You Will Learn
- Program structure (`main`, headers)
- Variables and primitive data types
- Input/output with `printf` and `scanf`

## Files
- `hello variables.c`
- `input output.c`

## Practice Goals
- Change values and re-run.
- Add one new variable of each type.
- Print formatted output with `%d`, `%f`, `%c`, `%s`.
