# Module Completion Checklist

Use this after each module.

- [ ] I read the module README fully.
- [ ] I compiled every `.c` file in the module.
- [ ] I changed at least 3 parts of code and re-ran.
- [ ] I can explain each function line-by-line.
- [ ] I solved one exercise without looking at examples.
- [ ] I fixed at least one input validation edge case.
