# Algorithms

This folder contains implementations of common algorithms in C, including sorting and searching algorithms.

## Contents

- **sorting/** - Various sorting algorithms (bubble sort, quick sort, merge sort, etc.)
- **searching/** - Search algorithms (binary search, linear search, etc.)
- **graph_algorithms/** - Graph traversal and algorithms
- **dynamic_programming/** - DP problems and solutions

Each algorithm includes:
- Source code implementation (.c files)
- Explanation and complexity analysis (.md files)
- Example usage
