# Debugging Techniques

This folder contains debugging techniques, tools, and examples for C programming.

## Contents

- **debugging_macros/** - Useful macros for debugging
- **memory_debugging/** - Memory leak detection and debugging
- **assertions/** - Using assertions for program validation
- **logging/** - Custom logging implementations
- **gdb_examples/** - GDB debugging session examples

## Techniques Covered

- Print debugging (printf statements)
- Using assert() for runtime checks
- Memory debugging with Valgrind
- GDB commands and breakpoints
- Common bug patterns and solutions
