# File Operations

This folder contains examples of file handling operations in C, including reading, writing, and file manipulation.

## Contents

- **text_files/** - Working with text files (reading, writing, appending)
- **binary_files/** - Binary file operations
- **csv_handling/** - CSV file parsing and generation
- **file_system/** - Directory operations and file system utilities

Each example demonstrates:
- File opening and closing
- Error handling
- Different file modes (r, w, a, rb, wb, etc.)
- Buffer management
