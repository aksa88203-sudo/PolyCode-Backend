# C Glossary

- `declaration`: introducing a name and type.
- `definition`: allocating storage or providing function body.
- `pointer`: variable storing memory address.
- `dereference`: access value pointed by pointer using `*`.
- `heap`: dynamic memory area (`malloc/free`).
- `stack`: automatic local variable storage.
- `segmentation fault`: invalid memory access crash.
- `undefined behavior`: program behavior with no C standard guarantee.
