# C Programming Learning Workspace

This repository is organized as a complete C learning system with guided modules, practice drills, and projects.

## Core Learning Modules

- `setup`
- `basics`
- `control flow`
- `functions arrays strings`
- `pointers memory`
- `structs files`
- `data structures algorithms`
- `recursion`
- `bitwise and cli`
- `file processing`
- `advanced topics`
- `mini projects`
- `capstone projects`

## Support Folders

- `study plan` - schedule and roadmap
- `checklists` - completion checklist
- `exercises` - practice drills
- `solutions` - reference solutions
- `templates` - reusable starter programs
- `tools` - helper scripts

## Recommended Flow

1. Learn one module.
2. Run all examples in that module.
3. Solve at least one file from `exercises`.
4. Compare with `solutions` if blocked.
5. Build mini/capstone projects.

## Build Everything

```bash
"tools/build all examples.sh"
```

Compiled outputs are saved in `build/`.

## Legacy Files

Older files like `01_io_statements.c` are preserved for compatibility, but the structured folders above are the primary learning path.
