# Topic Roadmap

## Foundation
- syntax, variables, operators
- stdin/stdout, format specifiers

## Control
- branching and loops
- validation and defensive checks

## Composition
- functions and modular design
- arrays, strings, and indexing discipline

## Memory
- stack vs heap
- pointer correctness
- ownership and `free`

## Structured Data
- structs, enums
- text and binary file handling

## Problem Solving
- recursion patterns
- sort/search patterns
- stack and queue basics

## Systems Style C
- bitwise operations
- command line arguments
- multi-file thinking and reusable templates
