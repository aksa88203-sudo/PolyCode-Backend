## Overview

JavaScript (JS) is a **high-level, interpreted, multi-paradigm programming language** that is one of the core technologies of the World Wide Web, alongside HTML and CSS. It is the only language that runs natively in web browsers, making it indispensable for web development.

```
┌────────────────────────────────────────────────┐
│             The Web Triad                       │
│                                                │
│   HTML  ──────►  Structure / Content           │
│   CSS   ──────►  Style / Presentation          │
│   JS    ──────►  Behavior / Interactivity       │
└────────────────────────────────────────────────┘
```

**Key Stats:**
- Created: 1995 by Brendan Eich at Netscape
- Paradigms: Procedural, Object-Oriented, Functional, Event-Driven
- Typing: Dynamic, Weakly typed
- Used by: ~98% of all websites

## History & Evolution

```
1995 ──► Brendan Eich creates "Mocha" in 10 days at Netscape
  │
1996 ──► Renamed to "JavaScript" (marketing near Java)
  │      Microsoft creates JScript for IE
  │
1997 ──► ECMAScript 1 standardized by ECMA International
  │
2009 ──► ES5 — strict mode, JSON support, Array methods
  │      Node.js created by Ryan Dahl (server-side JS!)
  │
2015 ──► ES6/ES2015 — MASSIVE update: classes, modules,
  │      arrows, promises, let/const, template literals
  │
2016+ ──► Annual releases: ES2016, ES2017... ES2024
  │
Today ──► Universal: browsers, servers, mobile, desktop,
          IoT, machine learning
```

## Core Characteristics

| Feature | Description |
|--------|-------------|
| **Interpreted** | Executed line by line (JIT compiled in modern engines) |
| **Dynamic Typing** | Variable types determined at runtime |
| **First-Class Functions** | Functions are values — passed, returned, assigned |
| **Prototype-Based OOP** | Inheritance via prototype chains |
| **Single-Threaded** | One call stack, non-blocking via event loop |
| **Garbage Collected** | Automatic memory management |
| **Multi-Paradigm** | Supports OOP, functional, procedural styles |
