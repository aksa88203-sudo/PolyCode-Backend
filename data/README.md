### 📋 Prerequisites
- No prior programming experience required
- Computer with Ruby 3.1+ installed
- Text editor or IDE (VS Code, RubyMine, etc.)

### 🔧 Installation

#### Windows
```bash
# Download and install RubyInstaller from https://rubyinstaller.org/
# Choose version 3.1.0 or higher
```

#### macOS
```bash
# Using Homebrew (recommended)
brew install ruby

# Or using rbenv for multiple versions
brew install rbenv
rbenv install 3.1.0
rbenv global 3.1.0
```

#### Linux
```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install ruby-full

# CentOS/RHEL
sudo yum install ruby
# or
sudo dnf install ruby
```

### ✅ Verify Installation
```bash
ruby --version
# Should show Ruby 3.1.0 or higher

gem --version
# Should show gem version
```

## 📝 Learning Path

### 🌱 Phase 1: Beginner (1-2 weeks)
1. **[Introduction](docs/01-introduction.md)** - Ruby philosophy & basics
2. **[Basic Syntax](docs/02-basic-syntax.md)** - Variables, operators, expressions
3. **[Data Types](docs/03-data-types.md)** - Numbers, strings, arrays, hashes
4. **[Control Flow](docs/04-control-flow.md)** - Conditionals, loops, iterators
5. **[Methods](docs/05-methods.md)** - Defining and calling methods

### 🚀 Phase 2: Intermediate (2-3 weeks)
6. **[Object-Oriented Programming](docs/06-oop.md)** - Classes, inheritance, polymorphism
7. **[Modules & Mixins](docs/07-modules.md)** - Modules, mixins, namespacing
8. **[Error Handling](docs/08-error-handling.md)** - Exceptions, rescue, ensure
9. **[File I/O](docs/09-file-io.md)** - Reading/writing files
10. **[Gems](docs/10-gems.md)** - Package management, using gems

### 🎓 Phase 3: Advanced (4-6 weeks)
11. **Choose your specialization:**
    - 🏗️ **Web Development** → Rails, Sinatra, APIs
    - 🗄️ **Database Programming** → SQL, ActiveRecord, optimization
    - 🔧 **DevOps** → CI/CD, containers, infrastructure
    - 🤖 **Machine Learning** → Data processing, algorithms
    - 🌐 **Networking** → Sockets, HTTP, distributed systems
    - ⚡ **Performance** → Optimization, profiling
    - 🔒 **Security** → Authentication, encryption
    - 🧪 **Testing** → TDD, BDD, automation

### 🏆 Phase 4: Expert (Ongoing)
- Build real projects
- Contribute to open source
- Master advanced topics
- Stay updated with Ruby ecosystem
