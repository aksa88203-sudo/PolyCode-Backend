# Source Code Directory

This directory contains all Python source code organized by category and purpose.

## 📁 Directory Structure

```
src/
├── 🧮 algorithms/          # Algorithm implementations
├── 🏗️ data_structures/     # Data structure implementations  
├── 🎮 games/               # Game-related projects
├── 🧮 mathematics/         # Mathematical computations
├── 🚀 projects/            # Complete application projects
└── 🛠️ utilities/           # Utility scripts and tools
```

## 🚀 Quick Start

Each script can be run independently. Navigate to the desired category and run:

```bash
# Example: Run an algorithm
cd algorithms/
python binary_search.py

# Example: Run a project
cd projects/
python budget_app.py
```
