# Ruby Guide Assets

This directory contains supporting assets for the Ruby Programming Guide, including diagrams, images, and other multimedia resources.

## Directory Structure

```
assets/
├── README.md           # This file
├── diagrams/          # Technical diagrams and flowcharts
│   ├── README.md       # Diagram descriptions
│   ├── ruby-object-model.png
│   ├── rails-request-lifecycle.png
│   └── gem-dependency-graph.png
├── images/             # Screenshots and illustrations
│   ├── README.md       # Image descriptions
│   ├── ruby-logo.png
│   ├── rails-console.png
│   └── code-examples/
└── templates/          # Document templates
    ├── README.md
    ├── project-template.md
    └── code-review-template.md
```

## Usage Guidelines

### Adding New Assets
1. Choose appropriate subdirectory
2. Use descriptive filenames
3. Optimize image sizes for web display
4. Update relevant README files
5. Reference assets in main documentation

### File Naming Conventions
- Use lowercase with hyphens: `ruby-object-model.png`
- Include version numbers when needed: `rails-7.0-architecture.png`
- Use descriptive names: `user-authentication-flow.png`

### Image Optimization
- PNG for diagrams and screenshots
- JPEG for photographs
- SVG for scalable graphics
- Keep file sizes under 500KB when possible
