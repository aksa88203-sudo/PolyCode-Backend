### createCompactList Function (Lines 17-38)
```cpp
int **createCompactList(int **grid, int rows, int cols, int *&rowSizes)
{
    rowSizes = new int[rows];
    for (int i = 0; i < rows; i++)
    {
        rowSizes[i] = countNonZero(grid[i], cols);
    }
    int **compact = new int *[rows];
    for (int i = 0; i < rows; i++)
    {
        compact[i] = new int[rowSizes[i]];
        int idx = 0;
        for (int j = 0; j < cols; j++)
        {
            if (grid[i][j] != 0)
            {
                compact[i][idx++] = grid[i][j];
            }
        }
    }
    return compact;
}
```

- **Line 19**: Allocate array to store row sizes
- **Line 20-22**: Count non-zero elements for each row
- **Line 24**: Allocate outer array of row pointers
- **Line 25-36**: Process each row
- **Line 27**: Allocate compact row with exact size needed
- **Line 28**: Initialize index for compact row
- **Line 29-35**: Copy only non-zero elements to compact row
