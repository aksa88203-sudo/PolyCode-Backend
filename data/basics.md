# C++ Basics

## What is C++?
C++ is a powerful programming language based on C. It supports both procedural and object-oriented programming and is widely used for software, games, and system applications.

## Key Features
- Fast and efficient
- Supports object-oriented programming
- Close to hardware (low-level control)
