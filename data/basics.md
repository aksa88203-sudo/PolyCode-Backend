# Rust Basics

## What is Rust?
Rust is a modern programming language focused on safety and performance. It helps prevent common programming errors like memory issues.

## Key Features
- Memory safety without garbage collection
- Fast and efficient
- Strong type system

    println!("Hello, World!");
