# Ruby Examples

This directory contains practical examples and implementations that demonstrate various Ruby programming concepts, patterns, and best practices. Each subdirectory focuses on a specific aspect of Ruby development.

## Directory Structure

```
examples/
├── README.md                    # This file
├── advanced-examples/           # Advanced Ruby concepts and patterns
├── challenges/                   # Programming challenges and solutions
├── creative-projects/            # Creative Ruby projects and applications
├── interview-prep/               # Interview preparation materials
├── projects/                     # Complete project examples
└── workshops/                    # Hands-on workshops and tutorials
```

## Getting Started

### Prerequisites
- Ruby 3.0 or higher
- Basic understanding of programming concepts
- Text editor or IDE
- Terminal/command line access

### Running Examples
Each example includes instructions on how to run it. Generally:

```bash
# Navigate to the example directory
cd examples/[category]/[specific-example]

# Run the Ruby file
ruby example_file.rb

# Or run tests if available
rspec test_file.rb
```

## Dependencies

Most examples use only Ruby standard library. Some examples may require additional gems:

```ruby
# Check for Gemfile in the example directory
bundle install
```

Common dependencies:
- `rspec` for testing
- `json` for data serialization
- `net/http` for HTTP requests
- `yaml` for configuration

## Testing

Many examples include test suites. To run tests:

```bash
# Run all tests in an example
rspec

# Run specific test file
rspec test/specific_test.rb

# Run with coverage
rspec --format documentation
```

## Troubleshooting

### Common Issues
1. **Ruby Version**: Ensure you're using Ruby 3.0+
2. **Dependencies**: Install required gems with `bundle install`
3. **Permissions**: Make sure files have execute permissions
4. **Encoding**: Use UTF-8 encoding for all files
