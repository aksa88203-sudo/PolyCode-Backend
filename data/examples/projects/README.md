# Ruby Projects

This directory contains practical Ruby projects that demonstrate real-world applications of the concepts covered in the Ruby guide.

## Project Structure

Each project is a self-contained example that showcases specific Ruby concepts and best practices.

- Data persistence

## How to Use These Projects

1. **Navigate to the project directory**
   ```bash
   cd ruby-guide/examples/projects/todo_list_manager
   ```

2. **Read the project README**
   Each project has its own README.md with specific instructions.

3. **Run the project**
   ```bash
   ruby main.rb
   ```

4. **Experiment and modify**
   Feel free to modify the code and experiment with different approaches.
