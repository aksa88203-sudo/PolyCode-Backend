# C Learning Guide: From Scratch to Advanced

This folder is your complete roadmap for C programming.

## How To Use This Guide

1. Follow modules in order.
2. Read the module `README.md` first.
3. Run every `.c` example.
4. Modify the code and re-run.
5. Complete mini projects without copy-pasting.

## Module Order

1. `setup`
2. `basics`
3. `control flow`
4. `functions arrays strings`
5. `pointers memory`
6. `structs files`
7. `data structures algorithms`
8. `advanced topics`
9. `mini projects`

## Compile and Run (important with spaces in file names)

```bash
gcc "hello variables.c" -o hello_variables
./hello_variables
```

From repository root, use full quoted path:

```bash
gcc "learning-guide/basics/hello variables.c" -o hello_variables
./hello_variables
```

## Weekly Study Plan (Suggested)

- Week 1: `setup`, `basics`, `control flow`
- Week 2: `functions arrays strings`, `pointers memory`
- Week 3: `structs files`, `data structures algorithms`
- Week 4: `advanced topics`, `mini projects`

## Skill Targets

By the end, you should be able to:
- Write clean C programs with functions and arrays.
- Use pointers safely with dynamic memory.
- Read/write files and model data with structs.
- Implement basic data structures and searching/sorting.
- Build small command-line C projects end-to-end.
